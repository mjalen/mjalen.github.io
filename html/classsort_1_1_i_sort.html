reference_sort ( corners)
static

Sort the reference objects in the order {UL, UR, DR, DL} or {UL, DR}.

See IImage.tangent_sort() for the method used in sorting.

Parameters
cornersThe corners of each image. Can be either a tensor of shape (2, 1, 4, 2) (for 2 objects) or (4, 1, 4, 2) (for 4 objects).
Returns
A tensor of the same shape as the input, but of the correct order.

◆ tangent_sort()

tangent_sort ( normal_corners,
dir0 = np.array([-1, -1]) )
static

Sorts an array of corners with respect to a given direction (vector).

The corners are sorted in increasing order by the angle from the corner to the given direction (vector)

Parameters
normal_cornersThe corners normalized by the approximate size of the image.
dir0The direction criteria to determine sorting order. The angle of each corner is found relative to this vector.
Returns
A numpy array of the index order of that corners should follow. See np.argsort() for an example of what to expect for an output.
