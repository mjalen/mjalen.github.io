@section description_corner_measure Description
 Program used to detect and measure windows from image by using multiple reference objects. Supports 2 and 4 reference objects.

@section libraries_corner_measure Libraries/Modules
- OpenCV: For image processing and aruco reference image detection.
- numpy: For array/tensor processing
- pandas: For data compilation and output. Used to construct a csv containing measure results and relative errors.
