Detailed Description

An interface of static methods for finding various parameters of a window image.

Done this way to avoid side-effects.

◆ get_window_corners()

get_window_corners ( corners)
static

Return the window corners from a 4 reference object image by extracting the corresponding reference object corner.

For instance, the UL corner extracts the UL corner of the UL reference object, UR corner extracts the UR corner of the UR reference object, etc.

Parameters
cornersA tensor of the shape (4, 1, 4, 2), containing the sorted reference objects and their corners.
Returns
A tensor of shape (4, 2), corresponding to the window corners in the order {UL, UR, DR, DL}.

Sorts an array of corners with respect to a given direction (vector).

The corners are sorted in increasing order by the angle from the corner to the given direction (vector)
